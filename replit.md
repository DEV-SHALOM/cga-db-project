# Chosen Generation Academy

## Overview
A React-based web application for Chosen Generation Academy, featuring authentication via Firebase.

## Tech Stack
- React 19 with Vite 7
- Tailwind CSS 4
- Firebase for authentication
- React Router for navigation

## Project Structure
- `src/` - Source code
  - `components/` - Reusable UI components
  - `pages/` - Page components
  - `hooks/` - Custom React hooks
  - `reports/` - Report-related modules
  - `utils/` - Utility functions
  - `firebase.js` - Firebase configuration
  - `AuthProvider.jsx` - Authentication context provider

## Development
Run the development server:
```bash
npm run dev
```
The app runs on port 5000.

## Build
```bash
npm run build
```
